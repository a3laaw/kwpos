# 📖 أدلة المستخدمين — نظام KWPOS

هذا المجلد يحتوي على **أدلة مستخدمين خطوة بخطوة** لكل دور وظيفي في النظام. كل دليل مكتوب بلغة عربية بسيطة، خالٍ من المصطلحات البرمجية، مناسب للموظفين الذين لم يستخدموا النظام من قبل.

## 🚀 كيف تبدأ؟

1. **اعرف دورك** — اسأل مديرك إذا لم تكن متأكداً.
2. **افتح دليلك** من الجدول أعلاه.
3. **ابدأ من القسم الأول** (تسجيل الدخول) حتى لو كنت تعتقد أنك تعرف — هناك تفاصيل مهمة.
4. **اتبع الخطوات بالترتيب** — كل مهمة مبنية على ما قبلها.
5. **راجع قسم "أسئلة شائعة"** عند مواجهة أي مشكلة.
6. **راجع قسم "ما لا تستطيع فعله"** لتعرف حدود صلاحياتك.

---

## ❓ أسئلة شائعة عبر كل الأدوار

| السؤال | الإجابة |
|--------|---------|
| نسيت كلمة المرور؟ | تواصل مع مدير النظام |
| لا أرى زراً معيناً؟ | قد لا يكون ضمن صلاحياتك — راجع دليلك |
| المنتج لا يظهر في POS؟ | تأكد أن لديه مخزون > 0 ومخزن مفعّل |
| الفاتورة لم تُطبع؟ | تأكد من طابعة الإيصالات + اتصالها بالكمبيوتر |
| السعر ظهر خطأ؟ | الأسعار يحددها المدير/المالك من شاشة التسعير |
| كيف أغير اللغة؟ | زر تبديل اللغة في أعلى الشاشة |
| كيف أغير العملة؟ | المالك/مدير النظام فقط من الإعدادات |
| أين أرى نقاط ولاء العميل؟ | في شاشة العملاء (بعد التعديل الأخير) |

## 📝 ملاحظات

- جميع الأدلة مكتوبة بناءً على **الكود الفعلي** للنظام، لا على افتراضات.
- إذا وجدت ميزة مذكورة في الدليل لكنها لا تعمل، قد تكون:
  - تحتاج صلاحية أعلى (راجع قسم "ما لا تستطيع فعله")
  - تحتاج إعداداً مسبقاً (مثل طابعة الإيصالات)
  - ميزة جديدة لم تُنشر بعد على خادمك
- الأدلة تُحدّث مع كل تحديث للنظام.
