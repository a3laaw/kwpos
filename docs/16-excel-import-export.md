# 16 — الاستيراد والتصدير من Excel

> دليل تدريبي مبسّط لوظائف استيراد وتصدير ملفات Excel في نظام KWPOS. يشرح أنواع التصدير (منتجات، عملاء، مبيعات، موردين، قيود)، أعمدة كل نوع، رؤية عمود التكلفة حسب الدور، الاستيراد (منتجات وعملاء)، القالب التدريبي القابل للتنزيل، تنسيق الفئات المتداخلة، التعامل مع روابط الصور، حدود الملفات، والأخطاء المحتملة. مكتوب لغير المختصين تقنياً مع الإشارة للملفات المصدرية.

الملفات الرئيسية:

| الملف | دوره |
|---|---|
| `src/lib/excel.ts` | دوال مساعدة عامة: `exportToExcel`, `parseExcelFile`, `downloadTemplate` |
| `src/app/api/excel/export/route.ts` | API تصدير (5 أنواع) |
| `src/app/api/excel/import-products/route.ts` | API استيراد المنتجات |
| `src/app/api/excel/import-customers/route.ts` | API استيراد العملاء |
| `src/components/shared/excel-buttons.tsx` | مكوّن أزرار التصدير/الاستيراد |
| `src/components/shared/export-toolbar.tsx` | شريط تصدير عام (PDF + Excel) لتقارير الصفحات |

## 2. أزرار Excel في الواجهة (Excel Buttons)

الملف: `src/components/shared/excel-buttons.tsx`. يقدّم مكوّنين:

### 2.1 زر التصدير `ExcelExportButton`

| الخاصية | القيمة | ملاحظة |
|---|---|---|
| `type` | `"sales"` / `"products"` / `"journal"` / `"customers"` / `"suppliers"` | نوع البيانات |

### 2.2 زر الاستيراد `ExcelImportButton`

| الخاصية | القيمة |
|---|---|
| `type` | `"products"` أو `"customers"` |

الزر قائمة منسدلة (Dropdown) تحوّي خيارين:

| الخيار | الوظيفة |
|---|---|

يتميّز بكونه RTL-aware: العنوان في أعلى المنتصف، الجدول تحته. أسماء الأعمدة تأخذ عرضاً بين 12 و40 حرفاً تلقائياً.

## 3. تصدير المنتجات (Products)

الـ API: `GET /api/excel/export?type=products`. الصلاحية: `canManageProducts(role)` = `OWNER`, `ADMIN`, `MANAGER`, `WAREHOUSE`.

### الأعمدة (تتغيّر حسب الدور!)

عمود **سعر التكلفة** يُعرض فقط للأدوار التي ترى التكلفة (`canSeeCost` = `OWNER`, `ADMIN`, `MANAGER`, `ACCOUNTANT`, `WAREHOUSE`). أمّا `SALES` و`CASHIER` فلا يرونه في ملف Excel.

| ترتيب | العمود | ملاحظة |
|---|---|---|
| 1 | الاسم | اسم المنتج |
| 2 | الباركود | قد يكون فارغاً |
| 3 | الفئة | **بالمسار الكامل** «أب > ابن > حفيد» |
| 4 | الكمية | رقم صحيح |
| 5 | حد الطلب | `reorderLevel` |
| 6 | سعر التكلفة | **مخفي** للـ SALES/CASHIER |
| 7 | سعر البيع | |
| 8 | الوحدة | نص حر مثل «قطعة» |
| 9 | رابط الصورة | URL كامل أو فارغ |

### بناء مسار الفئة

يستخدم الـ API دالة `buildCategoryPathMap` التي تتبع سلسلة `parentId` صعوداً لتبني المسار الكامل. مثال:

- فئة «عطور رجالية» تحت «عطور» → تُصدَّر كـ `عطور > عطور رجالية`
- فئة «عطور» بدون أب → تُصدَّر كـ `عطور` فقط

> هذا التنسيق يطابق تنسيق الاستيراد، فيمكنك تصدير المنتجات، تعديلها، ثم إعادة استيرادها دون فقدان بنية الفئات.

اسم الملف الناتج: `products.xlsx`. اسم الورقة: `الأصناف`.

## 8. استيراد المنتجات (Import Products)

الـ API: `POST /api/excel/import-products` (multipart/form-data، حقل `file`). الصلاحية: `ADMIN`, `WAREHOUSE` فقط.

### الأعمدة المطلوبة / الاختيارية

| العمود | مطلوب | ملاحظة |
|---|---|---|
| **الاسم** | ✅ | إن فارغ يُتجاوز الصف |
| الباركود | اختياري | يُستخدم في المطابقة |
| الفئة | اختياري | يدعم التنسيق المتداخل |
| الكمية | اختياري (افتراضي 0) | رقم |
| حد الطلب | اختياري (افتراضي 5) | رقم |
| سعر التكلفة | اختياري (افتراضي 0) | رقم |
| سعر البيع | اختياري (افتراضي 0) | رقم |
| الوحدة | اختياري (افتراضي «قطعة») | يجب أن تتطابق مع وحدة موجودة في الإعدادات لربط `unitId` |
| رابط الصورة | اختياري | يدعم روابط مباشرة وGoogle Drive وImgur |

> الكود يقبل أيضاً الأسماء الإنجليزية للأعمدة (مثل `name`, `barcode`, `category`) كبدائل.

### قاعدة المطابقة (Update vs Create)

1. إن وُجد منتج بنفس الباركود → يُحدَّث.
2. وإلا إن وُجد بنفس الاسم → يُحدَّث.
3. وإلا → يُنشأ منتج جديد.

الحقول المُحدَّثة للمنتج الموجود: `barcode`, `categoryId`, `quantity`, `reorderLevel`, `costPrice`, `salePrice`, `unit`, `unitId`, `imageUrl` (إن وُجد).

### تنسيق الفئات المتداخلة «أب > ابن»

يدعم الاستيراد بناء شجرة فئات بعمق غير محدود عبر الفاصل `>`:

| الإدخال | النتيجة |
|---|---|
| `عطور` | يستخدم فئة «عطور» أو يُنشئها كفئة جذر |
| `عطور > عطور رجالية` | يستخدم/يُنشئ «عطور» كأب ثم «عطور رجالية» كابن |
| `عطور > عطور رجالية > فاخرة` | ثلاث مستويات (أب → ابن → حفيد) |

خوارزمية `resolveOrCreateCategory` (في `import-products/route.ts`):

1. تقسيم النصّ بـ `>` وقصّ الفراغات.
2. لكل جزء: البحث في الذاكرة المؤقتة، فإن لم يوجد فالبحث في قاعدة البيانات بـ `parentId + name`.
3. إن لم يوجد إنشاؤه تحت الأب المناسب.
4. تُحدَّث الذاكرة المؤقتة لتسريع الصفوف اللاحقة.

### التعامل مع روابط الصور

العمود `رابط الصورة` يدعم ثلاثة أنواع من الروابط:

| النوع | المثال | المعالجة |
|---|---|---|
| رابط مباشر | `https://example.com/images/p1.jpg` | يُخزَّن كما هو |
| مشاركة Google Drive | `https://drive.google.com/file/d/FILE_ID/view` | يُحوَّل إلى `https://drive.google.com/uc?export=view&id=FILE_ID` |
| Google Drive `open?id=` | `https://drive.google.com/open?id=FILE_ID` | يُحوَّل إلى نفس الصيغة أعلاه |
| Imgur | `https://i.imgur.com/abc123.jpg` | يُخزَّن كما هو (رابط مباشر) |

> الكود يفحص وجود `drive.google.com` أو `open?id=` في الرابط، ويستخرج `FILE_ID` عبر regex، ويبني رابطاً مباشراً قابلاً للعرض في `<img>`. باقي الروابط تُخزَّن كما هي.

### الوحدات

الوحدة المُدخَلة في عمود `الوحدة` تُطابَق بالاسم مع الوحدات الموجودة في الإعدادات (`src/app/api/units/route.ts`). إن طابأت وحدة موجودة → يُربط `unitId`. وإلا → يبقى `unitId = null` مع تخزين النصّ في حقل `unit`.

### الردّ

```json
{
  "ok": true,
  "total": 120,
  "created": 80,
  "updated": 35,
  "skipped": 5,
  "catsCreated": 3,
  "errors": []
}
```

| الحقل | المعنى |
|---|---|
| `total` | إجمالي الصفوف في الملف |
| `created` | منتجات جديدة أُنشئت |
| `updated` | منتجات موجودة حُدِّثت |
| `skipped` | صفوف بلا اسم |
| `catsCreated` | فئات جديدة أُنشئت أثناء الاستيراد |
| `errors` | أخطاء لكل صف (تُقطَع إلى أول 10) |

## 11. حدود الملفات والتحقق الأمني

يُطبَّق التحقق نفسه على `import-products` و`import-customers`:

| الحدّ | القيمة | الكود |
|---|---|---|
| الحجم الأقصى | **5 ميجابايت** (5 × 1024 × 1024 بايت) | `file.size > 5 * 1024 * 1024` |
| الامتدادات المسموحة | `.xlsx`, `.xls`, `.csv` | `validTypes = [".xlsx", ".xls", ".csv"]` |
| الحد الأقصى للصفوف | **5000 صف** | `rows.length > 5000` |
| الملفات الفارغة | ممنوعة | `rows.length === 0` |

> هذه الحدود تحمي الخادوم من الملفات الضخمة أو الضارة. لاستيراد أكثر من 5000 منتج، قسّم ملفك إلى دفعات.

## 12. الأخطاء المحتملة

### أخطاء عامة (HTTP)

| الكود | الحالة | السبب |
|---|---|---|
| `unauthorized` | 401 | غير مسجَّل الدخول |
| `forbidden` | 403 | الدور غير مسموح |
| `no-file` | 400 | لم يُرفَع أي ملف |
| `invalid-file-type` | 400 | الامتداد ليس `.xlsx/.xls/.csv` |
| `file-too-large` | 413 | تجاوز 5 ميجابايت |
| `empty-file` | 400 | الملف بلا صفوف بيانات |
| `too-many-rows` | 400 | تجاوز 5000 صف (يُرجَع `max: 5000`) |
| `invalid-type` | 400 (تصدير) | `?type=` غير معروف |

### أخطاء الاستيراد (تُجمَع في `errors` ولا تُوقف العملية)

| الرسالة | السبب |
|---|---|
| `{name}: {error}` | فشل إنشاء/تحديث منتج أو عميل محدَّد (مثل خطأ في قاعدة البيانات) |
| – (صفوف تُجاوز بصمت) | صف بلا اسم في عمود «الاسم» |

### نصائح للاستيراد الناجح

| المشكلة | الحل |
|---|---|
| المنتجات تتكرَّر بدل التحديث | تأكد أن الباركود متطابق تماماً (المسافات، الأصفار البادئة) |
| الفئات لا تُربط | استخدم الفاصل ` > ` بمسافات، أو `>` بدون مسافات (كلاهما مقبول) |
| الوحدة لا تُربط بـ `unitId` | أضف الوحدة في شاشة الإعدادات أولاً بنفس الاسم |
| الصور لا تظهر | استخدم رابطاً مباشراً ينتهي بـ `.jpg` أو `.png`، أو حوّل روابط Google Drive |
| الملف مرفوض | تأكد أن الحجم ≤ 5MB والصفوف ≤ 5000 والامتداد صحيح |
| النصوص بالإنجليزية لا تُقرأ | الكود يقبل الأعمدة الإنجليزية (`name`, `barcode`...) كبديل، لكن العناوين العربية مفضّلة |

## 14. أسئلة شائعة

**س: هل أستطيع تصدير المنتجات بعمود التكلفة دائماً؟**
ج: لا. العمود يُخفى تلقائياً لأدوار `SALES` و`CASHIER` حتى لو حاولوا التصدير. هذا حماية مزدوجة (defense-in-depth) لمنع تسريب البيانات المالية عبر Excel.

**س: ماذا يحدث لو استوردت منتجاً بباركود موجود لكن باسم مختلف؟**
ج: يُحدَّث الباركود والاسم والكمية والأسعار. المطابقة بالباركود أولاً، فإن طابق → يُعتبر المنتج نفسه.

**س: هل أستطيع استيراد 10000 منتج دفعة واحدة؟**
ج: لا، الحد 5000 صف. قسّم ملفك إلى دفعتين.

**س: هل أستطيع تصدير القيود بفلاتر تاريخ؟**
ج: نعم، عبر `?from=YYYY-MM-DD&to=YYYY-MM-DD`. إن تركت الفلاتر فارغة يُصدَّر كل السجل.

**س: هل يدعم النظام استيراد الموردين أو الفواتير من Excel؟**
ج: لم يظهر في الكود أي API لاستيراد الموردين أو الفواتير. الاستيراد محصور في المنتجات والعملاء.

**س: ما الفرق بين `ExcelExportButton` و`ExportToolbar`؟**
ج: الأول يستدعي API خادومياً (`/api/excel/export`) ويصدّر بيانات من قاعدة البيانات. الثاني يصدّر بيانات موجودة في الذاكرة (مثل جدول في صفحة) مباشرةً من المتصفّح، ويضيف زر PDF.
